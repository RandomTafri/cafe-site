# Café Site (Next.js + Tailwind)

Static, deploy-ready café website with editable JSON content.

## Dev
```bash
npm install
npm run dev
```

## Deploy (Vercel)
- Push to GitHub
- Import repo on https://vercel.com → Deploy

## Edit Content
- `data/menu.json`
- `data/gallery.json`
- `data/reviews.json`
- `data/events.json`
- `data/settings.json`
